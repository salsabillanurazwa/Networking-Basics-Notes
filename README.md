# Ubuntu 26.04 LTS Installation Using VirtualBox

## Project Overview

This project documents the process of installing Ubuntu 26.04 LTS on Oracle VirtualBox as a virtual machine.

The purpose of this project is to gain practical experience in operating system installation, virtual machine configuration, Linux environment setup, and basic command-line operations.

## Objectives

- Install Ubuntu 26.04 LTS using Oracle VirtualBox
- Configure a virtual machine environment
- Understand basic Linux commands
- Manage files and permissions in Linux

## Software Used

- Oracle VirtualBox
- Ubuntu 26.04 LTS
- Windows Operating System
